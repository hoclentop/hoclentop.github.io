Nên viết hàm taoxau(s) để tạo xâu thuần nhất.

Hàm nenxau(s) để nén xâu, dùng mảng đếm

Thuật toán
hàm taoxau(string s)
+	string p="";
+	Với mỗi kí tự s[i]: 
  o	nếu s[i]>= 'A' và s[i] <= 'Z' thì p=p+s[i];
+	Nếu p="" thì p="AZ";
+	Trả về p

hàm nenxau(string s)
+	char c[256];
+ Fill mảng c = 0 hết;
+	Với mỗi kí tự s[i]: tăng c[s[i]];
+	string p="";
+ vơi mỗi kí tự x chạy từ 'A' đến 'Z': 
  - Nếu c[x]>0 thì p=p+x+to_string(c[x]);
+	Trả về p
