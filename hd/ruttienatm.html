Vì cần số tờ ít nhất nên ta ưu tiên trả tờ tiền mệnh giá lớn nhiều nhất.

Số tờ 500K là: s500 = n div 500000; (div là chia lấy thương)

Nếu s500 > 1000 thì s500 = 1000;

Khi đó, số tiền còn lại phải trả là: n = n - s500 * 500000;

--------------------------------------------

Số tờ 200K là: s200 = n div 200000;

Nếu s200 > 1000 thì s200 = 1000;

Khi đó, số tiền còn lại phải trả là: n = n - s200 * 200000;

--------------------------------------------

Số tờ 100K là: s100 = n div 100000;

Nếu s100 > 1000 thì s100 = 1000;

Khi đó, số tiền còn lại phải trả là: n = n - s100 * 100000;

Lập luận tiếp tục như vậy...

Thuật toán
+ Nhập n;
+ Nếu n < 50000 thì thông báo không thực hiện được và kết thúc
+ s500=min(n div 500000, 1000);
+ n = n - s500 * 500000;
+ s200=min(n div 200000, 1000);
+ n = n - s200 * 200000;
+ s100=min(n div 100000, 1000);
+ n = n - s100 * 100000;
+ s50=min(n div 50000, 1000);
+ n = n - s50 * 50000;
+ s20=min(n div 20000, 1000);
+ n = n - s20 * 20000;
+ s10=min(n div 10000, 1000);
+ n = n - s10 * 10000;
+ Nếu n > 0 thì thông báo không thực hiện được và kết thúc
+ Xuất s10+s20+s50+s100+s200+s500;
+ Xuất s10, s20, s50, s100, s200, s500;
