Nhận xét:

+ Thứ tự các bước đi là không quan trọng, VD LDUR, LRDU, DURL,... cũng như nhau.

+ Đi 1L và 1R sẽ về chỗ cũ.

+ Đi 1U và 1D sẽ về chỗ cũ.

⇒ Để quay về chỗ cũ thì số bước U bằng số bước D, số bước L bằng số bước R.

Gọi u, d, l, r tương ứng là số bước đi U, D, L, R.

Số bươc đi tối đa là min(l,r)*2 + min(u,d)*2.

//toi.hoclen.top
#include 
 
using namespace std;
int n,l=0,r=0,u=0,d=0;
string s;
 
int main()
{
    cin>>n>>s;
    for (int i=0; i<n; i++)
        if (s[i]=='U') u++;
        else if(s[i]=='D') d++;
        else if (s[i]=='L') l++;
        else r++;
    cout<<min(l,r)*2+min(u,d)*2;
    return 0;
}
