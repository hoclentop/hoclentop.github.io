Phân tích - Ý tưởng

Vì KQ là số lớn nhất nên ưu tiên chữ số đầu tiên là 9 (nếu s≥9), khi đó, phần còn lại là số lớn nhất có (n-1) chữ số và có tổng các chữ số là (s-9).

Nếu s<9 thì số đó là s00...0 (chữ số đầu là s còn lại là các số 0).

Thuật toán
+ Với mỗi i chạy từ 1 đến n:
    + Nếu s>=9 thì
        + Viết số 9;
        + s = s-9;
    + ngược lại thì:
        + viết số s;
        + s = 0;
Ghi chú - Bàn luận
Bình luận
Chương trình

//hoclen.top
#include <iostream>
using namespace std;
int main()
{
    int n, s;
    cin >> n >> s;
    if (n*9 < s)
        cout << -1;
    else
        for (int i = 1; i<=n; i++)
        {
            if (s >= 9)
            {
                cout << 9;
                s = s - 9;
            }
            else
            {
                cout << s;
                s = 0;
            }
        }
    return 0;
}
